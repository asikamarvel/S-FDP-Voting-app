# SFDP Voting Competition Guide

## How to Vote

| Platform | Action | Account |
|----------|--------|---------|
| Twitter | Retweet | @disease_society |
| YouTube | Comment | Society for Disease Prevention |
| Instagram | Comment | @societyfordiseaseprevention |
| Facebook | React | Society for Disease Prevention |

## Twitter
1. Follow @disease_society
2. Find the competition tweet
3. Retweet it

## YouTube
1. Subscribe to the channel
2. Find the competition video
3. Leave a comment

## Instagram
1. Follow @societyfordiseaseprevention
2. Find the competition post
3. Leave a comment

## Facebook
1. Like the page
2. Find the competition post
3. React (Like, Love, etc.)

## Rules
- One vote per action per post
- No fake accounts
- No spam
